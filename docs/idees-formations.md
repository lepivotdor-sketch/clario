## Convention de remplissage

Quand une formation est publiée :

1. Coche la case `[X]`.
2. Mets le titre exact en **gras**.
3. Indique le prix et le statut entre parenthèses.
4. Ajoute la formation dans `data/formations.json`.

Exemple :
```
- [x] **IA-002** : Écrire un courriel de vente clair avec l'IA *(15 $, publié)*
```
